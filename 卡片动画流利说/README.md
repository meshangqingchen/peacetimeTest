###编译配置
CocoaPods 是一款 Cocoa 项目包管理工具, 如果你尚未安装 Cocoapods, 运行以下命令进行安装:

	$ gem install cocoapods $ pod setup
下载源代码

	$ git clone https://github.com/poholo/MCScrollCardDemo
安装依赖

	$ pod install
打开 MCScrollCardDemo.xcworkspace

这就完了, 祝你玩得愉快! :beers:
